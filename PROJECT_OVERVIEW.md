## 🎨 设计规范

### 色彩系统

```
主色调（深靛蓝）: #1E3A8A
├─ 用途：标题、导航、主要按钮
├─ 占比：30%
└─ 传递：专业、可靠、信任

辅助色（亮橙）: #F97316
├─ 用途：强调、CTA、图标
├─ 占比：10%
└─ 传递：创意、活力、行动

中性色：
├─ 浅灰 #F3F4F6 (背景 - 60%)
├─ 中灰 #9CA3AF (辅助文字)
└─ 深灰 #1F2937 (正文)
```

### 字体系统

```
标题字体：Montserrat
├─ 字重：700-900
├─ 用途：H1, H2, H3, 按钮
└─ 特点：粗体、现代、视觉冲击

正文字体：Open Sans
├─ 字重：400-600
├─ 用途：段落、导航、表单
└─ 特点：清晰、易读、友好
```

### 排版规则

```
标题层级：
├─ H1: 3rem (48px)      - 首屏主标题
├─ H2: 2rem (32px)      - 分区标题
├─ H3: 1.4rem (22.4px)  - 卡片标题
└─ Body: 1rem (16px)    - 正文

行高：1.6 (正文)
字母间距：1px (CTA按钮)
对齐方式：左对齐（英文习惯）
```

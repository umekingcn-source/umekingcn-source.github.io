## ✏️ 如何添加您的内容

### 方法1: 直接编辑HTML（推荐）

打开 `index.html`，找到品牌故事部分（约第340-405行）：

```html
<!-- Brand Story Section -->
<section class="brand-story" id="about">
    ...
    <div class="brand-card-content">
        <p data-i18n="brand.card.intro">
            [品牌故事介绍文字 - 请在此添加您的内容] ← 替换这里
            We are dedicated to providing...
        </p>
        
        <p data-i18n="brand.card.mission">
            [使命愿景文字 - 请在此添加您的内容] ← 替换这里
            Our mission is to help your brand...
        </p>
    </div>
    ...
</section>
```

### 方法2: 编辑多语言文件

如果需要多语言支持，编辑 `js/language-manager.js`：

```javascript
// 英文
'brand.card.intro': '您的英文品牌故事...',
'brand.card.mission': '您的英文使命愿景...',

// 中文
'brand.card.intro': '您的中文品牌故事...',
'brand.card.mission': '您的中文使命愿景...',

// 其他语言同理
```

## 🖼️ 更换品牌图片

### 步骤1: 准备图片
- **推荐尺寸**: 800x600 像素或更大
- **格式**: JPG/PNG
- **风格建议**: 高质量产品照片、工厂照片、团队照片

### 步骤2: 上传图片
将图片放到 `images/` 目录，例如：`images/brand-story.jpg`

### 步骤3: 更新HTML
在 `index.html` 中找到：

```html
<div class="brand-story-image">
    <img src="images/hero-tag.jpg" alt="U-MEKING Brand Story" 
         onerror="this.src='images/case-patches.jpg'">
    <!-- 将 src 改为您的图片路径 -->
</div>
```

改为：

```html
<img src="images/brand-story.jpg" alt="U-MEKING Brand Story">
```

### 步骤4: 更新年份标签
```html
<div class="brand-year">Since 2010</div>
<!-- 改为您的创立年份 -->
<div class="brand-year">Since 2015</div>
```

## 🎨 自定义样式

### 修改颜色方案

编辑 `css/brand-story.css`：

#### 1. 更改背景色
```css
.brand-story {
    background: #0a0e1a;  /* 改为您喜欢的颜色 */
}
```

#### 2. 更改按钮颜色
```css
.brand-read-more {
    background: linear-gradient(135deg, #4ade80 0%, #22c55e 100%);
    /* 改为您的品牌色，例如紫色： */
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}
```

#### 3. 更改统计数据颜色
```css
.stat-number {
    background: linear-gradient(135deg, #4ade80 0%, #22c55e 100%);
    /* 改为您的品牌色 */
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}
```

### 修改字体大小

```css
/* 主标题 */
.brand-story-title {
    font-size: 8rem;  /* 改小一点：6rem */
}

/* 卡片标题 */
.brand-card-title {
    font-size: 2.5rem;  /* 改小一点：2rem */
}
```

## ✨ 特殊效果

### 1. 图片悬停放大
鼠标悬停在品牌图片上时，图片会缓慢放大

### 2. 卡片悬停效果
鼠标悬停在信息卡上时：
- 背景变亮
- 卡片上浮
- 边框加强

### 3. 按钮悬停动画
鼠标悬停"Learn More"按钮时：
- 颜色变深
- 向右移动
- 箭头图标向右滑动

### 4. 渐入动画
页面加载时：
- 标题从上方淡入
- 内容从下方淡入

```

- 企业使命

示例：
"我们的使命是通过卓越的工艺和创新设计，帮助每一个品牌讲述属于自己的
故事。我们坚信，每一个细节都能成就品牌的独特性，每一件产品都应该展现
最高的品质标准。"
```

## 🐛 常见问题

### Q1: 品牌故事部分不显示？
```
✓ 检查是否引入了 brand-story.css
✓ 清除浏览器缓存
✓ 检查浏览器控制台错误
```

### Q2: 图片显示不正确？
```
✓ 确认图片路径正确
✓ 确认图片文件存在
✓ 检查图片格式（支持JPG/PNG）
```

### Q3: 统计数据不对齐？
```
✓ 确认3个统计项都存在
✓ 检查CSS是否正确加载
✓ 在移动端是单列显示（正常）
```

### Q4: 多语言内容没有翻译？
```
✓ 确认添加了data-i18n属性
✓ 确认language-manager.js已更新
✓ 刷新页面重新加载脚本
```

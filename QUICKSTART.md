## ✏️ 自定义内容

### 1. 更改公司名称

**在 `index.html` 中查找并替换：**
- `LogoCustomPro` → 你的公司名称

**位置：**
- 导航栏 logo (第 23 行)
- 页脚 (第 496 行)
- 浏览器标签标题 (第 8 行)

### 2. 更新联系信息

**在 `index.html` 中修改：**

```html
<!-- 电话 (第 415 行) -->
<p>+1 (555) 123-4567</p>  <!-- 改为你的电话 -->

<!-- 邮箱 (第 424 行) -->
<p>info@logocustompro.com</p>  <!-- 改为你的邮箱 -->

<!-- WhatsApp (第 433 行) -->
<p>+1 (555) 123-4567</p>  <!-- 改为你的 WhatsApp -->

<!-- 地址 (第 451 行) -->
<p>123 Business Street, Suite 100<br>New York, NY 10001</p>
<!-- 改为你的地址 -->
```

### 3. 更改品牌颜色

**在 `css/styles.css` 中修改：**

```css
/* 第 30-32 行 */
:root {
    --primary-blue: #1E3A8A;    /* 主色调（深蓝） */
    --primary-orange: #F97316;   /* 辅助色（橙色） */
}
```

**改为你的品牌色：**
```css
:root {
    --primary-blue: #你的主色;
    --primary-orange: #你的辅助色;
}
```

💡 **提示：** 使用 [Adobe Color](https://color.adobe.com/) 或 [Coolors](https://coolors.co/) 生成配色方案

## 🚀 发布到互联网

### 最快的方法 - Netlify (免费)

1. **访问** https://netlify.com
2. **注册账号**（使用 Email 或 GitHub）
3. **拖拽整个项目文件夹** 到 Netlify 的上传区域
4. **等待 30 秒** ⏳

**免费包含：**
- ✅ 自动 HTTPS (SSL 证书)
- ✅ 全球 CDN 加速
- ✅ 无限流量
- ✅ 自动备份

## 📱 测试移动端

### 在浏览器中测试：

1. **Chrome DevTools：**
   - 按 `F12` 打开开发者工具
   - 点击 📱 图标（Toggle device toolbar）
   - 选择 iPhone / iPad / 其他设备
   - 查看移动端效果

2. **在真实设备上测试：**
   - 使用本地服务器（见下方）
   - 在同一 Wi-Fi 下，用手机访问你的电脑 IP 地址

### 启动本地服务器：

**方法 1 - Python (Mac/Linux/Windows):**
```bash
cd 项目文件夹路径
python -m http.server 8000
```
然后访问：http://localhost:8000

**方法 2 - Node.js:**
```bash
npx http-server
```

**方法 3 - VS Code 插件:**
- 安装 "Live Server" 插件
- 右键 `index.html` → "Open with Live Server"

## 🔧 常见问题快速解决

### Q1: 图片不显示？
**解决方案：**
1. 检查 `images/` 文件夹是否存在
2. 检查图片文件名是否正确（区分大小写）
3. 检查图片路径：应该是 `images/文件名.jpg`

### Q2: 样式不正常？
**解决方案：**
1. 检查 `css/styles.css` 文件是否存在
2. 清除浏览器缓存（Ctrl + F5 或 Cmd + Shift + R）
3. 检查浏览器控制台是否有错误（F12）

### Q3: 按钮不响应？
**解决方案：**
1. 检查 `js/main.js` 文件是否存在
2. 打开浏览器控制台（F12）查看是否有 JavaScript 错误
3. 确保使用现代浏览器（Chrome 90+, Firefox 88+, Safari 14+）

### Q4: 移动端菜单不工作？
**解决方案：**
1. 检查浏览器宽度是否 < 768px
2. 检查 JavaScript 是否正确加载
3. 清除缓存并刷新

### Q5: 表单提交后没反应？
**原因：** 表单目前仅有前端验证，未连接后端。

**解决方案：**
- 参考 `DEPLOYMENT.md` 中的"设置联系表单"部分
- 使用 EmailJS (推荐) 或其他表单服务

# Banner 特效方案B - 动态光效+浮动装饰

## 📋 方案概述

方案B在保留方案4原有的排版和内容基础上，添加了一系列动态特效，使Banner更加醒目和独特。

## ✨ 特效详情

### 1. 🌈 动态渐变背景（呼吸光效）
- **效果**：背景渐变色在深蓝色和金色之间循环切换
- **周期**：8秒缓动循环
- **实现**：使用 CSS `@keyframes gradientShift` 动画

### 2. 💡 文字发光霓虹效果
- **主品牌名"U-MEKING"**：
  - 多层发光阴影效果
  - 金色霓虹光晕随呼吸变化
  - 3秒循环动画
  
- **副标题"U-Meking IDEAS"**：
  - 柔和的金色光晕
  - 4秒循环动画
  - 与主标题形成错落感

### 3. 🎯 浮动几何装饰元素
- **6个浮动装饰形状**：
  - 圆形（带径向渐变）
  - 方形（带线性渐变，45度旋转）
  - 三角形（半透明金色）
  
- **动画效果**：
  - 上下左右浮动
  - 轻微旋转
  - 闪烁变化
  - 每个形状有独立的动画周期（15-22秒）

### 4. 📦 产品卡片悬浮动画
- **两个产品栏（GARMENT TAGS / PROMO PRODUCTS）**：
  - 持续的垂直浮动动画（6秒周期）
  - 交替浮动（延迟3秒）
  - Hover时：
    - 放大1.02倍
    - 向上移动8px
    - 显示渐变光边框
    - 金色发光阴影

### 5. ✅ 列表图标脉动效果
- **复选标记图标**：
  - 持续的缩放脉动（2秒周期）
  - 金色发光阴影
  - 三个图标错开动画（延迟0.3秒）
  - Hover时旋转10度并放大

### 6. 🎨 标题下划线光效
- **分隔线动画**：
  - 光晕呼吸效果（3秒周期）
  - 金色光束从左滑到右（2秒循环）
  
### 7. 📊 统计数据脉动
- **三个统计项**：
  - 背景和边框颜色变化
  - 金色光效脉动（4秒周期）
  - 交替动画（延迟1.3秒）
  - Hover时放大并增强光效

### 8. 🔘 CTA按钮特效
- **"Start Your Project"按钮**：
  - 持续的阴影脉动（3秒周期）
  - Hover时：
    - 放大1.08倍
    - 向上移动3px
    - 内部光波扩散效果
    - 增强的金色光晕
  - 箭头图标向右移动

### 9. 🖱️ 鼠标视差效果
- **品牌名和产品卡片**：
  - 跟随鼠标移动产生轻微位移
  - 营造3D深度感
  - 移出页面时平滑归位

### 10. 📜 滚动渐变效果
- **背景透明度**：
  - 随页面滚动逐渐降低背景亮度
  - 让下方内容更突出

## 🔧 自定义调整

如需调整特效强度，可以修改以下参数：

```css
/* 动画速度 */
animation-duration: 3s; /* 增大=变慢，减小=变快 */

/* 发光强度 */
text-shadow: 0 0 20px rgba(202, 138, 4, 0.8); /* 增大第3个值=更亮 */

/* 浮动幅度 */
transform: translateY(-10px); /* 增大负值=浮动更高 */

/* 缩放程度 */
transform: scale(1.02); /* 增大值=放大更多 */
```

## 🎬 动画性能

- 使用 `will-change` 属性优化性能
- 优先使用 `transform` 和 `opacity` 实现动画
- 使用 `requestAnimationFrame` 优化鼠标事件
- 所有动画均使用 `ease-in-out` 缓动函数

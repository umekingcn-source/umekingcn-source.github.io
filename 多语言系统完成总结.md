## 🔧 技术实现细节 | Technical Details

### 架构设计

```
┌─────────────────────────────────────┐
│     LanguageManager (Core)          │
├─────────────────────────────────────┤
│  - 语言检测                         │
│  - 翻译管理                         │
│  - 本地存储                         │
│  - UI更新                           │
└─────────────────────────────────────┘
           │
           ├─────► 自动语言检测
           │       (navigator.language)
           │
           ├─────► LocalStorage
           │       (持久化存储)
           │
           ├─────► DOM操作
           │       (data-i18n属性)
           │
           └─────► RTL支持
                   (dir属性 + CSS)
```

### 数据结构

```javascript
{
    languages: {
        'en': { name, nativeName, flag, dir },
        'zh': { ... },
        // ...
    },
    translations: {
        'en': { 'key': 'value', ... },
        'zh': { 'key': '值', ... },
        // ...
    }
}
```

### HTML集成

```html
<!-- 引入样式 -->
<link rel="stylesheet" href="css/language-switcher.css">

<!-- 添加翻译标记 -->
<h1 data-i18n="hero.title">U-MEKING</h1>

<!-- 引入脚本 -->
<script src="js/language-manager.js"></script>
```

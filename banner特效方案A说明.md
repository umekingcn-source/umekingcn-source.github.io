# Banner 特效方案A - 简约高端+微动画

## 📋 方案概述

方案A在保留方案4原有的排版和内容基础上，采用简约优雅的设计理念，通过细腻的微动画和精致的交互反馈，营造专业、高端的品牌形象。

## ✨ 特效详情

### 1. 🎨 三段式静态渐变背景
- **效果**：深蓝 → 中蓝 → 金色的平滑渐变
- **特点**：稳定、专业，无动画干扰
- **配色**：
  - 起点：`rgba(15, 23, 42, 0.92)` 深蓝
  - 中点：`rgba(30, 41, 59, 0.88)` 中蓝  
  - 终点：`rgba(202, 138, 4, 0.85)` 金色

### 2. 💡 文字细腻阴影增强

**主品牌名"U-MEKING"**：
- 经典深色阴影 + 金色微光边缘
- 字间距：3px → 4px（悬停时）
- Hover时轻微增强金色光晕
- 无循环动画，仅响应交互

**副标题"U-Meking IDEAS"**：
- 金色文字配深色阴影
- 颜色：金色 → 亮金色（悬停时）
- 增加白色微光边缘

### 3. 📏 智能下划线
- **初始**：150px宽，金色渐变
- **悬停**：自动扩展到200px
- **光晕**：静态柔和金色阴影
- **过渡**：0.5秒平滑动画

### 4. 📦 产品卡片优雅交互

**两个产品栏设计**：
- 半透明磨砂玻璃背景
- 细边框 + 深色阴影
- **顶部装饰线**：
  - 初始隐藏
  - Hover时从顶部出现金色渐变线

**悬停效果**：
- 向上移动5px
- 背景稍微增亮
- 边框变为金色
- 增强阴影深度
- 添加金色外发光边框

**过渡**：使用 `cubic-bezier(0.4, 0, 0.2, 1)` 缓动函数

### 5. 🎯 标题渐进下划线

**"GARMENT TAGS" / "PROMO PRODUCTS"**：
- 金色标题文字
- 底部边框：白色半透明
- **装饰线**：
  - 初始：30%宽度的金色实线
  - Hover：扩展至100%
  - 过渡：0.4秒流畅动画
- **字间距**：2px → 3px（卡片悬停时）

### 6. ✅ 列表图标简洁缩放

**复选标记图标**：
- 金色图标 `rgba(202, 138, 4, 1)`
- 无循环动画，保持静态
- Hover时：
  - 缩放至1.15倍
  - 颜色变亮至 `rgba(234, 179, 8, 1)`
  - 列表项文字向右移动0.5rem
  - 文字透明度增加

### 7. 📊 统计数据简约悬停

**三个统计项**：
- 半透明白色背景
- 细白色边框
- Hover时：
  - 向上移动2px
  - 背景变为金色半透明
  - 边框变为金色
  - 增加阴影

**无循环动画**，仅响应用户交互

### 8. 🔘 CTA按钮精致过渡

**"Start Your Project"按钮**：
- 金色渐变背景
- 金色边框
- 静态阴影（无脉动）

  - 反转渐变方向

### 9. 🚫 无浮动装饰元素

- 保持页面简洁专业

### 10. 🚫 无鼠标视差效果

## 🔧 自定义调整

如需调整特效强度，可以修改以下参数：

```css
/* 悬停移动距离 */
transform: translateY(-5px); /* 减小值=移动更少 */

/* 过渡速度 */
transition: all 0.3s; /* 增大=变慢，减小=变快 */

/* 阴影强度 */
box-shadow: 0 8px 25px rgba(0, 0, 0, 0.3); /* 调整第2、3个值控制范围和模糊 */

/* 背景透明度 */
background: rgba(255, 255, 255, 0.12); /* 第4个值越大=越不透明 */
```

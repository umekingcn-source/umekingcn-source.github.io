## 🔧 开发者快速集成

### 为新页面添加多语言

#### 1. 引入文件
```html
<!-- 在 <head> 中 -->
<link rel="stylesheet" href="css/language-switcher.css">

<!-- 在 </body> 前 -->
<script src="js/language-manager.js"></script>
```

#### 2. 添加翻译标记
```html
<h1 data-i18n="hero.title">U-MEKING</h1>
<p data-i18n="hero.description">Your text here</p>
<a data-i18n="nav.catalog">Catalog</a>
```

#### 3. 完成！
刷新页面，语言切换器自动出现！

## 📝 添加新翻译

### 步骤 1: 编辑 `language-manager.js`

找到 `this.translations` 对象：

```javascript
this.translations = {
    'en': {
        'your.new.key': 'Your English Text'  // ← 添加这里
    },
    'zh': {
        'your.new.key': '你的中文文本'  // ← 添加这里
    },
    // ... 其他语言
}
```

### 步骤 2: 在HTML中使用

```html
<div data-i18n="your.new.key">Your English Text</div>
```

### 步骤 3: 刷新测试

打开页面，切换语言，验证翻译是否正确。

## 🐛 常见问题速查

### Q: 语言切换器不显示？
```
✓ 检查是否引入了 language-switcher.css
✓ 检查是否引入了 language-manager.js
✓ 打开浏览器控制台查看错误
```

### Q: 某些文字没有翻译？
```
✓ 确保元素有 data-i18n 属性
✓ 确保翻译键在所有语言中都存在
✓ 检查翻译键名是否正确
```

### Q: 阿拉伯语布局混乱？
```
✓ 检查 language-switcher.css 是否正确加载
✓ 确认有 body.rtl 相关样式
✓ 清除浏览器缓存重试
```

### Q: 移动端语言切换器太小？
```
✓ 在移动端会自动只显示旗帜图标
✓ 这是正常的响应式设计
✓ 可以在 CSS 中调整 @media 断点
```

# Multi-Language System User Guide

## 📋 概述 | Overview

U-MEKING网站现已支持5种语言的完整国际化功能：

- 🇺🇸 **English** (英语)
- 🇨🇳 **中文** (Chinese Simplified)
- 🇩🇪 **Deutsch** (德语)
- 🇪🇸 **Español** (西班牙语)
- 🇦🇪 **العربية** (阿拉伯语)

---

## ✨ 功能特性 | Features

### 1. 自动语言检测
- 系统会自动检测用户的浏览器语言设置
- 首次访问时自动切换到最匹配的语言
- 支持的语言代码：`en`, `zh`, `de`, `es`, `ar`

### 2. 语言持久化
- 用户选择的语言会保存在浏览器本地存储中
- 下次访问时自动加载上次选择的语言
- 无需重新选择

### 3. RTL（从右到左）支持
- 完整支持阿拉伯语的从右到左（RTL）排版
- 自动调整布局、图标和导航方向
- 专门优化的阿拉伯语字体显示

### 4. 美观的语言切换器
- 位于导航栏右侧
- 显示国旗图标和语言名称
- 支持悬停和点击动画效果
- 移动端优化显示

### 5. 即时切换
- 点击语言按钮后立即切换
- 无需刷新页面
- 平滑的过渡动画

## 🚀 快速开始 | Quick Start

### 使用语言切换器

1. **桌面端**：
   - 在页面顶部导航栏右侧找到语言切换器
   - 点击想要切换的语言按钮
   - 页面内容立即切换

2. **移动端**：
   - 打开移动菜单
   - 滚动到底部找到语言切换器
   - 点击语言按钮切换

## 🛠️ 技术实现 | Technical Implementation

### 文件结构

```
js/
  └── language-manager.js      # 多语言管理核心
css/
  └── language-switcher.css    # 语言切换器样式
```

### HTML集成

在HTML中，需要翻译的元素使用 `data-i18n` 属性标记：

```html
<h1 data-i18n="hero.title">U-MEKING</h1>
<p data-i18n="hero.description">Custom Garment Tags & Promotional Products</p>
<a data-i18n="nav.catalog">Catalog</a>
```

### JavaScript API

```javascript
// 获取当前语言
languageManager.currentLanguage

// 切换语言
languageManager.changeLanguage('zh')

// 获取翻译
languageManager.t('hero.title')
```

## 📝 添加新的翻译 | Adding New Translations

### 1. 在 `language-manager.js` 中添加翻译键

```javascript
this.translations = {
    'en': {
        'your.new.key': 'Your English Text'
    },
    'zh': {
        'your.new.key': '您的中文文本'
    },
    'de': {
        'your.new.key': 'Ihr deutscher Text'
    },
    'es': {
        'your.new.key': 'Tu texto en español'
    },
    'ar': {
        'your.new.key': 'النص الخاص بك بالعربية'
    }
};
```

### 2. 在HTML中使用

```html
<div data-i18n="your.new.key">Your English Text</div>
```


所有RTL相关的样式都在 `body.rtl` 选择器下：

```css

    text-align: right;
```

## 🔧 常见问题 | FAQ

### Q1: 如何添加新语言？

在 `language-manager.js` 中：

1. 添加到 `this.languages` 对象：
```javascript
'fr': { 
    name: 'French', 
    nativeName: 'Français',
    flag: '🇫🇷', 
    dir: 'ltr'
}
```

2. 添加翻译到 `this.translations` 对象
3. 刷新页面即可使用

### Q2: 语言切换后为什么有些文字没变？

- 确保该元素添加了 `data-i18n` 属性
- 确保翻译键在所有语言中都有定义
- 检查翻译键名是否正确

### Q3: 如何禁用自动语言检测？

修改 `language-manager.js`：

```javascript
this.currentLanguage = 'en'; // 强制默认英语
// 注释掉：this.currentLanguage = this.getSavedLanguage() || this.detectLanguage();
```

### Q4: 移动端语言切换器不显示？

- 检查是否引入了 `language-switcher.css`
- 确认 `language-manager.js` 正确加载
- 检查浏览器控制台是否有错误

### Q5: 如何自定义语言检测逻辑？

修改 `detectLanguage()` 方法：

```javascript
detectLanguage() {
    const userLang = navigator.language || navigator.userLanguage;
    
    // 添加您的自定义逻辑
    if (userLang.startsWith('zh')) return 'zh';
    if (userLang.startsWith('de')) return 'de';
    
    return 'en'; // 默认语言
}
```

## 🚨 注意事项 | Important Notes

### 1. 性能优化
- 语言切换是即时的，不会重新加载页面
- 翻译数据存储在内存中，切换快速
- 建议定期清理未使用的翻译键

### 2. SEO考虑
- 目前使用客户端渲染切换语言
- 建议为SEO添加 `<html lang="xx">` 属性
- 可考虑使用服务器端渲染（SSR）

### 3. 浏览器兼容性
- 支持所有现代浏览器
- IE11需要添加polyfills
- 移动浏览器完全支持

### 4. 本地存储
- 使用 `localStorage` 保存语言偏好
- 清除浏览器数据会重置语言选择
- 隐私模式下每次需要重新选择

## 🔧 自定义选项

### 修改收件人邮箱

在 `js/quick-contact.js` 中找到：
```javascript
const mailtoLink = `mailto:evelyn@u-meking.com?subject=${subject}&body=${body}`;
```

改为您的邮箱：
```javascript
const mailtoLink = `mailto:your@email.com?subject=${subject}&body=${body}`;
```

### 修改颜色主题

在 `css/quick-contact.css` 中修改：
```css
.quick-submit-btn {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    /* 改为您的品牌色 */
}
```

### 修改表单字段

在 `index.html` 的弹窗部分添加或删除字段：
```html
<div class="quick-form-group">
    <label for="phone">电话</label>
    <input type="tel" id="phone" name="phone">
</div>
```

## 🐛 常见问题

### Q1: 邮件客户端没有打开？
```
A: 这是正常的，部分浏览器会拦截mailto链接。
   用户可以：
   1. 检查浏览器弹窗拦截设置
   2. 手动发送邮件到 evelyn@u-meking.com
   3. 使用底部的完整联系表单
```

### Q2: 表单提交后没有反应？
```
A: 检查：
   1. 是否填写了所有必填字段
   2. 邮箱格式是否正确
   3. 浏览器控制台是否有错误
   4. JavaScript文件是否正确加载
```

### Q3: 弹窗样式不正常？
```
A: 确认：
   1. quick-contact.css 已正确引入
   2. CSS文件路径正确
   3. 没有其他CSS冲突
   4. 清除浏览器缓存
```

### Q4: 在手机上显示不好？
```
A: 这个已经优化过响应式，如果还有问题：
   1. 清除缓存重试
   2. 检查viewport设置
   3. 测试不同手机尺寸
```
